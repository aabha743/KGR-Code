FIRST PROGRAM!

Get Started with C++ with your first program!
This program is a basic print program.

    Objectives
  • Learn how to give an output in C++
  • Learn more about the basic syntax of C++
    About The various commands used
  • #include iostream where iostream is enclosed between <>. It tells the compiler to include the standard iostream file which contains declarations of all the standard input/output library functions. It is a preprocessor along with a library.
  • using namespace std This is used to import the entirety of the std namespace into the current namespace of the program.
  • int main() This line is used to declare a function named “main” which returns data of integer type.
  • coutThis tells the compiler to display the message enclosed in double quotes.
  • return 0This returns 0 value to the main function.
    Steps to Print Hello World
  1. Start by including the preprocessor.
  2. Next, add the namespace std and end the line with a semicolon.
  3. Call the int main function now and open new curly braces or flower brackets.
  4. Within the curly braces, write the cout<<"Hello World"; command.
  5. Return 0 to the main function and close the curly braces.
  6. Click on the execute button and run your code.
