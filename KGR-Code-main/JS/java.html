FIRST PROGRAM!

Get Started with Java with your first program!
This program is a basic print program.

    Objectives
  • Learn how to use the print function in Java
  • Learn more about the basic syntax of Java
    About The commands used
  • public class In Java, every program starts with the declaraton of a class and the corresponding class name. The keyword public is used to tell the compiler that the class is a public class.
  • public static void main(String[] args) { ... } This is the main method. Every application in Java must contain the main method. The Java compiler starts executing the code from the main method.
  • System.out.println("Hello, World!"); The code above is a print statement. It prints the text Hello, World! to standard output.
    Steps to Print Hello World
  1. Declare a class and a class name of your choice.
  2. Open curly braces, move to the next line and declare the main method. Again open new curly braces.
  3. Move to the next line and use the print statement mentioned above. Close all the curly braces.
  4. Click on execute and check the results!
